# Tanner's Tech News - Visual Assets

This directory contains all visual assets for the Tanner's Tech News blog.

## SVG Assets (Vector - Scalable)

| File | Purpose | Usage |
|------|---------|-------|
| `logo.svg` | Main logo (512x512) | Website header, branding |
| `logo-icon.svg` | Icon-only logo (100x100) | Favicon, small icons |
| `favicon.svg` | Browser favicon (32x32) | Browser tabs, bookmarks |
| `hero-banner.svg` | Hero banner (1920x400) | Homepage header banner |
| `social-preview.svg` | Social sharing (1200x630) | OpenGraph, Twitter Cards |

### Logo Design
The robot logo features:
- **Blue eye** (#3b82f6) - represents technology/trust
- **Orange eye** (#f97316) - represents energy/creativity
- **Green waveform mouth** (#10b981) - represents data/audio processing
- **Sleek modern head** with antenna - unmistakably AI/tech themed
- **Circuit pattern background** - reinforces tech theme

### Color Scheme
- Primary Blue: `#3b82f6` (tech, trust)
- Accent Orange: `#f97316` (energy, creativity)
- Success Green: `#10b981` (data, activity)
- Dark Navy: `#0f172a` / `#1e293b` (backgrounds)

## Usage Instructions

### In HTML/Website
```html
<!-- Main Logo -->
<img src="images/logo.svg" alt="Tanner's Tech News" width="512">

<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="images/favicon.svg">

<!-- Hero Banner -->
<div class="hero" style="background-image: url('images/hero-banner.svg')">

<!-- Social Preview -->
<meta property="og:image" content="https://dano311.github.io/tanners-tech-news/images/social-preview.svg">
```

### For Thumbnails
```html
<!-- Article thumbnail -->
<img src="images/openai-regulation.png" alt="AI Regulation">
```
